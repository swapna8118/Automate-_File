# Reciprocity Python Task 4 – Automate File Renaming

## 📌 Project Overview

This project is a Python automation script that renames all files in a specified folder using a sequential naming pattern such as:

* file_1.txt
* file_2.jpg
* file_3.pdf

The script preserves each file's original extension while assigning a new sequential filename.

## 🎯 Objective

* Learn Python file handling.
* Practice using the `os` module.
* Understand basic automation with Python.

## 🛠️ Technologies Used

* Python 3
* os module

## ✨ Features

* Reads all files from a folder.
* Renames files automatically.
* Preserves original file extensions.
* Skips directories.

## ▶️ How to Run

1. Clone this repository.
2. Open the Python file.
3. Update the `folder_path` variable with your folder location.
4. Run the script:

   ```bash
   python file_renamer.py
   ```

## 📂 Example

Before:

```
photo.png
notes.txt
resume.pdf
```

After:

```
file_1.png
file_2.txt
file_3.pdf
```
